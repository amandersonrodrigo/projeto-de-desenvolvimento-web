Seletorem em CSS

Aplique CSS para diferentes partes do HTML

1. O seletor de elementos direciona elementos com base no nome da tag HTML.

  1. Seletores de classe direcionam elementos com base no valor do atributo de classe.
  2. Seletores de ID direcionam elementos com base no valor do atributo id.
  3. Seletores de atributo direcionam elementos com base em seus atributos e valores.
  4. O seletor universal direciona todos os elementos.
